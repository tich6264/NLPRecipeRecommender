# NLPRecipeRecommender
Recommend recipes using NLP techniques including word embeddings and NER.

## Folder Structure

* NER
    * BERT
        * Training
    * spaCy
        * data_preprocessing
        * training
            * configs
            * data
* Initial_database
    * SQLite
* label_experiment

* tfidf_search
    *  Recipe search using TF-IDF (not used in final project)
* WordEmbeddings
    * Data prep
    * Model comparison
    * Ground truth metrics
